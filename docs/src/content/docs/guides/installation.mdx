<Steps>

1. Install AstroUI

   ```bash
   npm install @beanwei/astro-ui
   # or
   pnpm add @beanwei/astro-ui
   # or
   yarn add @beanwei/astro-ui
   ```

2. Configure the tailwind

   ```js title="src/tailwind.config.mjs"
    import astroui from '@beanwei/astro-ui/tailwind' // ðŸ‘ˆ-Step1

    /** @type {import('tailwindcss').Config} */

        './node_modules/@beanwei/astro-ui/src/**/*.{astro,ts}', // ðŸ‘ˆ-Step3
        './src/**/*.{astro,html,js,jsx,md,mdx,svelte,ts,tsx,vue}',

   ```

</Steps>
